컴닥터 상세 소개

AI가 내 PC를 진단하고, 가장 효율적인 업그레이드를 알려드립니다

컴닥터란?

컴닥터는 Windows PC의 사양을 자동으로 수집하여, 사용자의 목적과 예산에 맞는 최적의 업그레이드 부품을 AI가 추천해주는 데스크톱 앱입니다.

복잡한 PC 지식 없이도, 몇 번의 클릭만으로 "게임을 위해 뭘 사야 하는지", "50만원으로 어떤 부품을 바꿔야 하는지" 바로 알 수 있습니다.

주요 기능

  • 자동 사양 수집: CPU, 메인보드, RAM, GPU, 저장장치, OS를 자동으로 읽어옵니다
  • 8가지 사용 목적: 게임, 영상 편집, 방송, 개발, 문서, 감상 등 목적별 최적화
  • 예산 슬라이더: 5만원~100만원 범위에서 예산 설정
  • AI 업그레이드 추천: Gemini Flash AI가 호환성·우선순위·가격을 동시에 분석
  • 호환성 검사: 메인보드 소켓/칩셋 기준으로 실제 장착 가능 여부 확인
  • 수동 입력 지원: 다른 PC의 사양도 직접 입력해서 진단 가능

수집하는 PC 정보

부품수집 항목방식

시스템 요구사항

  • 운영체제: Windows 10 / 11 (64-bit)
  • 인터넷 연결 필요 (AI 분석 시)
  • 별도 드라이버 설치 불필요

개인정보 처리

진단 시 수집되는 PC 사양 정보는 AI 분석 목적으로만 서버에 전송됩니다. 개인 식별 정보는 수집하지 않습니다.
